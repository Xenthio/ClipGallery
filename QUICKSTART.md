# Quick Start Guide

## Installation

1. Make sure you have Node.js installed (v16 or higher)
2. Clone the repository:
   ```bash
   git clone https://github.com/Xenthio/ClipGallery.git
   cd ClipGallery
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
4. Run the application:
   ```bash
   npm start
   ```

## First Time Setup

1. **Add Directories**
   - Click **Settings** in the left sidebar
   - Click **Add Directory**
   - Select folder(s) containing your game clips
   - Example locations:
     - Windows: `C:\Users\YourName\Videos\GameName`
     - Linux: `~/Videos`
     - macOS: `~/Movies`

2. **Scan for Clips**
   - Click **Scan Clips** button in the top toolbar
   - Wait for scan to complete
   - Clips will appear in the gallery

## Daily Usage

### Browse Clips
- **All Clips**: Click "All Clips" in sidebar
- **By Game**: Click game name in sidebar
- **By Tag**: Click tag name in sidebar
- **Favorites**: Click "Favorites" for 4â˜…+ clips

### Watch a Clip
1. Click any clip thumbnail
2. Video opens in player
3. Use standard video controls

### Rate Clips
- **From Gallery**: See stars below each clip (visual only)
- **From Player**: Click stars (1-5) to set rating

### Tag Clips
1. Open clip in player
2. Type tag name in input box
3. Press Enter or click Add Tag
4. Click tag to remove it

### Find Clips
- **Search**: Type in search box
- **Filter by Game**: Use game dropdown
- **Filter by Rating**: Use rating dropdown

### Edit Clips
1. Open clip in player
2. **Trim**: Click "Trim Clip", enter start/end times
3. **Export**: Click "Export as H.264" to convert

## Tips

- Rate clips as you watch them
- Use consistent tag names
- Organize clips in game-specific folders
- Run periodic scans to catch new clips
- 4+ star rating = Favorites filter

## Common Locations

### NVIDIA ShadowPlay
- `C:\Users\[You]\Videos\[Game]` (Windows)

### OBS Studio
- `C:\Users\[You]\Videos` (Windows)
- `~/Videos` (Linux)
- `~/Movies` (macOS)

### Steam Screenshots
- `C:\Program Files (x86)\Steam\userdata\[ID]\760\remote\[AppID]\screenshots` (Windows)
- `~/.local/share/Steam/userdata/[ID]/760/remote/[AppID]/screenshots` (Linux)

## Supported Formats

### Video Files
`.mp4`, `.mkv`, `.avi`, `.mov`, `.webm`, `.m4v`, `.flv`

### Codecs
AV1, H.265/HEVC, H.264, VP9, VP8
